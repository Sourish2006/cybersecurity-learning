# 🔄 TCP vs UDP

## 📘 TCP (Transmission Control Protocol)
- Reliable
- Connection-based
- Slower but accurate

## 📘 UDP (User Datagram Protocol)
- Fast
- No connection
- Less reliable

## 🧠 My Understanding:
TCP is used when accuracy is important (like web browsing), while UDP is used when speed is important (like streaming).
