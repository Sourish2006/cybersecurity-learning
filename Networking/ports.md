# 🔌 Ports in Networking

## 📘 What is a Port?
A port is a communication endpoint used by devices to send and receive data.

## 🔑 Common Ports:
- 80 → HTTP
- 443 → HTTPS
- 22 → SSH
- 21 → FTP

## 🧠 My Understanding:
Ports help identify which service is running on a system. Open ports can be a security risk.
