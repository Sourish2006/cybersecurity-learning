# Firewall

## 📚 Introduction

A firewall is a network security device or software that monitors and controls incoming and outgoing network traffic based on predefined security rules.

Firewalls act as a barrier between trusted internal networks and untrusted external networks, helping to protect systems from unauthorized access and cyber threats.

---

## 🔑 Why Firewalls are Important

Firewalls help to:

* Prevent unauthorized access
* Protect networks from malicious traffic
* Enforce security policies
* Monitor network communication
* Reduce the risk of cyber attacks

---

## 🔄 How Firewalls Work

Firewalls inspect network packets and determine whether they should be allowed or blocked based on configured rules.

A firewall can:

* Allow trusted traffic
* Block suspicious traffic
* Filter packets based on IP addresses, ports, and protocols

---

## 📌 Types of Firewalls

### Packet Filtering Firewall

Examines packets based on source IP address, destination IP address, ports, and protocols.

---

### Stateful Firewall

Tracks active connections and makes decisions based on the connection state.

Advantages:

* More secure than packet filtering firewalls
* Better traffic monitoring

---

### Stateless Firewall

Examines each packet independently without considering previous packets.

---

### Proxy Firewall

Acts as an intermediary between clients and servers.

Advantages:

* Hides internal network addresses
* Provides additional security

---

### Next Generation Firewall (NGFW)

Provides advanced security features such as:

* Deep Packet Inspection (DPI)
* Intrusion Prevention System (IPS)
* Application awareness
* Malware protection

---

## 🌐 Real-World Applications

Firewalls are commonly used in:

* Home networks
* Enterprise networks
* Data centers
* Cloud environments
* Corporate infrastructures

---

## 🛡 Advantages of Firewalls

* Improved network security
* Traffic monitoring and filtering
* Protection against unauthorized access
* Enforcement of security policies
* Reduced attack surface

---

## 🔑 Key Takeaways

* A firewall monitors and filters network traffic.
* Firewalls protect systems from unauthorized access.
* Stateful firewalls track active connections.
* NGFWs provide advanced security capabilities.
* Firewalls are essential components of modern network security.
