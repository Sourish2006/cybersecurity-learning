# 🌐 Router

## 📘 What is a Router?
A router is a networking device that connects multiple networks and forwards data between them.

## 🔑 Key Functions:
- Connects different networks (LAN ↔ Internet)
- Routes data packets
- Assigns IP addresses (via DHCP)

## 📡 Example:
- Home WiFi router connects devices to the internet

## 🔄 How it works:
- Receives data packets
- Checks destination IP
- Sends data to correct network

## 🧠 My Understanding:
A router acts like a traffic manager that directs data between networks, which is essential for internet communication.
