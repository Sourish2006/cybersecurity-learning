# Switching Concepts

## 📚 Introduction

Switching is the process of forwarding frames within a local area network (LAN). Network switches use MAC addresses to determine where frames should be sent.

Switches improve network performance and efficiency by reducing unnecessary traffic.

---

## 🔑 Why Switching is Important

Switching helps:

* Connect devices within a LAN
* Improve network performance
* Reduce collisions
* Enable efficient communication

---

## 🌐 What is a Switch?

A switch is a Layer 2 network device that forwards frames based on MAC addresses.

Common devices connected to switches include:

* Computers
* Servers
* Printers
* Access Points

---

## 📋 MAC Address Table

Switches maintain a MAC address table containing:

* MAC addresses
* Associated ports

This allows switches to forward frames only to the intended destination.

---

## 🔄 Switching Process

1. A frame arrives at a switch.
2. The switch examines the destination MAC address.
3. It checks the MAC address table.
4. The frame is forwarded to the appropriate port.

---

## 🔀 Switch vs Hub

### Switch

* Intelligent device
* Uses MAC addresses
* Reduces collisions
* Better performance

### Hub

* Broadcasts frames to all devices
* Less efficient
* More collisions

---

## 🛡 Advantages of Switching

* Increased network efficiency
* Reduced collisions
* Better performance
* Improved scalability

---

## 🔑 Key Takeaways

* Switches operate at Layer 2.
* Switches use MAC addresses for forwarding.
* MAC address tables enable efficient communication.
* Switches provide better performance than hubs.
