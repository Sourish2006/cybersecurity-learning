# 🌐 OSI Model

## 📘 What is OSI Model?
OSI (Open Systems Interconnection) model is a framework that explains how data travels through a network in 7 layers.

## 🔑 7 Layers of OSI:
1. Physical → Hardware, cables
2. Data Link → MAC address
3. Network → IP address
4. Transport → TCP/UDP
5. Session → Connection management
6. Presentation → Data format, encryption
7. Application → User interface (HTTP, FTP)

## 🧠 My Understanding:
OSI model helps understand how data moves step by step in a network, which is important for troubleshooting and security.
