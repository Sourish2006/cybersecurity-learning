# SSH (Secure Shell)

## 📚 Introduction

SSH (Secure Shell) is a network protocol used to securely access and manage remote systems over a network.

It provides encrypted communication and is widely used by system administrators and cybersecurity professionals.

---

## 🔑 Why SSH is Important

SSH helps:

- Secure remote access
- Encrypt communications
- Prevent eavesdropping
- Enable secure file transfers
- Support authentication using keys

---

## 🌐 Default Port

SSH uses the following default port:

22

---

## 🔄 How SSH Works

1. A client initiates a connection.
2. The SSH server responds.
3. Authentication is performed.
4. A secure encrypted session is established.

---

## 🔐 Authentication Methods

### Password Authentication

Users authenticate using a username and password.

### Key-Based Authentication

Uses:

- Public Key
- Private Key

Key-based authentication is more secure than passwords.
