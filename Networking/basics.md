# 🌐 Networking Basics

## 📘 What I Learned:
- Networking means communication between systems  
- IP address is used to identify devices  

## 🔑 Key Concepts:
- Devices communicate using networks  
- Each device has a unique IP address  

## 🧠 My Understanding:
Networking is the foundation of cybersecurity because all communication between systems happens through networks.
