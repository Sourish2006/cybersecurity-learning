# 🌐 DNS (Domain Name System)

## 📘 What is DNS?
DNS converts domain names (like google.com) into IP addresses.

## 🌳 DNS Hierarchy:
DNS works in a hierarchical structure:

- Root Server → Top level (.)
- TLD Server → (.com, .org, .net)
- Domain Server → (example.com)

👉 Flow:
User → Root → TLD → Domain → IP Address

---

## 📑 DNS Records:

- A Record → Maps domain to IPv4 address  
- AAAA Record → Maps domain to IPv6 address  
- CNAME → Domain alias (redirect to another domain)  
- MX → Mail server record  
- NS → Name server record  

---

## 🔑 Key Points:
- Humans use domain names  
- Computers use IP addresses  
- DNS acts like a translator  

---

## 🧠 My Understanding:
DNS is essential for internet communication, and its hierarchy helps efficiently resolve domain names step by step.
