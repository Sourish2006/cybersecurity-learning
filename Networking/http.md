# 🌐 HTTP & HTTPS

## 📘 What is HTTP?
HTTP is a protocol used to transfer data between client and server.

## 🔐 HTTPS:
- Secure version of HTTP
- Uses encryption (SSL/TLS)

## 🔑 Difference:
- HTTP → Not secure
- HTTPS → Secure

## 🧠 My Understanding:
HTTP is used for communication, but HTTPS ensures data is encrypted and secure.

## 🔄 Client-Server Communication

- Client → Browser (user side)
- Server → Website server

👉 Flow:
Client → Request → Server  
Server → Response → Client

## 📩 HTTP Request:
- Sent by client to server
- Example: GET /index.html

## 📤 HTTP Response:
- Sent by server to client
- Contains data (HTML, status code)

## 🧠 My Understanding:
HTTP works on a request-response model where the client asks for data and the server responds with it.
